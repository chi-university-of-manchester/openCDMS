There are five kinds of subprojects here:

plugin-xxx   These are XDoclet2 plugins (and JavaDoc tags if they are used by that plugin only).
testapp-xxx  These contain Java source code used by some of the plugin-xxx' tests.
sample-xxx   These are samples for the users, with focus on demonstration.
tags-xxx     These are JavaDoc @tags that are used by more than one plugin. (none yet).
reference    Generates documentation and updates the website.
